## Problem
No centralized view of all active Claude projects. Status, progress,
and priorities are scattered across individual GitHub repos.

## Target User
Solo developer (zowen22) managing multiple Claude-assisted projects.

## Goals
A lightweight, always-accessible dashboard that shows all projects
at a glance — status, progress, and summary — without manual maintenance.

## Core Features
- Display all registered Claude project repos
- Show status and % complete per project pulled from .md files
- Auto-updates on GitHub push
- Accessible on mobile and desktop via GitHub Pages URL

## Constraints
- Must use public GitHub repos and GitHub API (no auth token)
- Single static HTML file — no build tools or frameworks
- Minimal maintenance overhead
