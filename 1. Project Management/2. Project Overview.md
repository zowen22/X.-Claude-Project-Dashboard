## Summary
A static GitHub Pages dashboard that reads project status from registered
Claude project repos via the GitHub API. Displays per-project status,
progress %, and summary in a clean minimalist card layout.

## Scope
**In:**
- Single index.html, no frameworks or build tools
- GitHub API reads from public repos (no auth token)
- Parses Project Overview.md for status/summary
- Parses Work Packages.md for progress % via checkbox count
- Mobile and desktop responsive

**Out:**
- Private repo support
- Authentication / GitHub token handling
- Backend or database of any kind
